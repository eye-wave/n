# ru~~n~~ - Use the Right Script Runner

**ru** is a CLI tool that automatically selects the correct script runner (like `npm`, `yarn`, `pnpm`, or `bun`) based on your project setup. It simplifies running scripts in environments where different tools are used. No need to remember which project uses what â€“ just run `ru`!

## Installation

Clone this repository and run install.sh

```bash
git clone repo:url
```
```bash
./install.sh
```

## Usage

```bash
ru
```
